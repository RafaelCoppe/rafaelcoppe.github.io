Gestion des sauvegardes

Compétences mises en œuvre
  • 1 - Gérer le patrimoine informatique
Contexte du Projet

Ce projet porte sur l'utilisation du logiciel Cobian Backup, un logiciel qui permet de programmer des sauvegardes des fichiers de l'ordinateur de manière régulière.
Les 3 types de sauvegardes sont :

  • Différentielles - La première sauvegarde est complète. Mais par la suite, le système sauvegarde tous les changements depuis la dernière sauvegarde complète
  • Incrémentielles - La sauvegarde initiale est complète et chaque sauvegarde suivante stocke les modifications apportées depuis la dernière sauvegarde
  • Complète - Chaque fois qu’un système est sauvegardé, tous les fichiers sont copiés

Adoptez une architecture MVC en PHP
＂Vous avez déjà fait vos premiers pas en PHP... Félicitations, il est maintenant temps d'apprendre à marcher pour de bon !＂

Le modèle MVC (Modèle-Vue-Controlleur) est un modèle d'architecture d'applications qui fonctionne en 3 étapes :

  • Modèle - Contient les données à afficher
  • Vue - Contient les modèles graphique des pages
  • Controlleur - Agit comme un intermédiaire entre le modèle et la vue
Cette architecture est utlisé par beaucoup de frameworks comme Symfony, Laravel ou AngularJS. Ici, la certification s'effectue en PHP, le 4ème langage web le plus utilisé avec HTML, CSS et JavaScript, et un des langages les plus utilisés avec le modèle MVC.
Pour mettre en pratique ces connaissances, on propose de développer un blog avec notamment la possibilité d'insérer des commentaires et de le convertir ensuite à la programmation orienté objet (POO).

Référencement d'un site web

Compétences mises en œuvre
  • 3 - Développer la présence en ligne de l’organisation
Contexte du Projet

L'agence Linkretz désire augmenter la visibilité de son site web et s'intérésse donc au réferencement. Elle demande à avoir une analyse rapide du référencement.
Pour améliorer le référencement du site Linkretz, on peut :

Calculer le page rank du site pour savoir où concentrer nos efforts
Le page rank représente la popularité d’une page d’un site web. On peut donc ainsi savoir si une page présente un problème qui diminuerait sa popularité par exemple
Optimiser les utilisations des balises
L’algorithme de Google analyse les balises HTML utilisées pour un site web pour le référencer. On peut donc améliorer son référencement en améliorant son utilisation des balises utiles au SEO comme ‘Title’ ou ‘Robot’ ainsi que l’en-tête du site
Ajouter un plan du site visible facilement
Afficher un plan du site web est une bonne manière de guider les utilisateurs sur le site, ce qui améliore la popularité des pages les moins bien redirigées et donc la popularité générale du site
Payer la visibilité sur certains mots clés via Google Ads
Google propose de payer pour mettre en avant son site web dans un certain mot-clé. Cela se passe sous formé d'enchères et permet de grandement augmenter la visibilité du site en augmentant le nombre d'utilisateurs tombant au hasard sur celui-ci
Inclure du texte alternatif sur vos images
Le texte alternatif d’une image (le texte qui s'affiche si l’image ne peut être chargée) est très important pour les moteurs de recherche car ils lui permettent de savoir ce que contient l’image
Choisir les bons mots-clé
Choisir les bons termes sur son site permet de mieux le référencer car le site ressortira durant les recherches des potentiels clients

Contexte du projet

Mon projet de stage a porté sur le scraping en Python :

Le scraping (grattage en français, parfois appelé harvesting) est une technique de développement qui consiste à créer un bot capable de se déplacer automatiquement sur un site web et récupérer les informations qui y sont affichées afin de les réutiliser.
Dans l'exemple "public" que j'ai effectué afin de remplir mon rapport de stage, j'ai récupéré les données d'un fichier qui liste les pistes d'atterissage pour avions dans le monde.
Chaque information récolté est enregistrée dans un fichier texte de type XML puis inséré dans une base de données. Grâce à ce fichier, j'ai pu créer une vérification des doublons afin d'éviter la redondance des informations.
