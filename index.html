Contexte du Projet

Cette application permet à un cinéma de rentrer les données de fréquentation d'un film sur un mois et une année précise Les données sont stockées dans un fichier de type XML, permettant une consultation des informations du cinéma
Pour entrer les information d'un film, le cinéma doit fournir :

  • Le nom du film
  • Le mois et année en question
  • Le nombre de séances
  • Le nombre de places enfant vendues
  • Le nombre de places adulte vendues
L'application permet ensuite plusieurs options afin de visualiser les données présentes selon des paramètres, comme le mois ou le nombre minimal de places enfants vendues

Contexte du Projet

Ce TP traite de la création d'utilisateur en SQL et de la gestion de leurs accès.
On travaille pour le groupe hôtelier Jenati qui est en train de développer un site web et qui a donc une base de données à gérer. On nous demande donc de vérifier si les commandes permettant de créer les comptes sont correctes. Les comptes à créer sont :

  • lecteur
  • admin_hotel
  • dpo
  • garden
  • dsi
Une fois les comptes crées, on doit modifier le compe garden pour lui enlever l'accès à une table et le compte dsi pour lui interdire de faire des ajouts dans la base de données.
