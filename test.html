Contexte du Projet

Le groupe pharmaceutique GSB désire contrôler davantage l'origine des frais de remboursement des déplacements de ses employés.
De plus, GSB désire gérer séparement les demandes de remboursement liées :

  • aux déplacements chez les médécins pour présenter des produits;
  • aux participations à des congrès;
  • aux participations à des formations obligatoires;
Ces trois dernières améliorations ont étés séparées parmi les 3 membres du groupe de travail puis mises en commun via GitHub

  • Catégorie : Application Client Lourd (Modèle en couches)
  • Langages : C# (Windows Forms)
  • Outils : VS Code
Contexte du Projet

Le groupe pharmaceutique GSB désire controller ses actions de mécénat.
L'application permet d'effectuer les 4 actions principales (Ajout, modification suppression et consultation) sur :

  • les associations;
  • les actions menées;
  • les partenariats;
Elle permet aussi d'ajouter et de consulter les limites d'argent de mécénat imposées aux associations.
